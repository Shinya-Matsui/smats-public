# AI Agent Skills Collection

このディレクトリには、AIエージェント（Claude Code Agent, Antigravityなど）に特定のタスクを遂行させるための「スキル定義ファイル (`SKILL.md`)」を格納しています。

## 概要
「スキル」とは、AIに対して特定の役割、手順、出力形式を事前に定義したマークダウンファイルです。これを読み込ませることで、汎用的なAIを「特定のタスクの専門家」として振る舞わせることができます。

## 収録スキル

### 1. Daily Reporter (日報作成)
- **場所**: [daily-reporter/SKILL.md](./daily-reporter/SKILL.md)
- **機能**: ユーザーの話を聞き取り、構造化されたMarkdown形式の日報を自動作成します。
- **使い方**: エージェントに対して「日報スキルを使って」と指示するだけで、ヒアリングからファイル作成までをガイドします。
