# Claude Code でプロフィールサイトを自動生成した話

<!-- category: AI/LLM -->

## 学んだこと
- **Claude Code**: Anthropic が提供する CLI ベースの AI コーディングアシスタント。ターミナルから直接コードの生成・編集・実行ができる。VS Code 拡張としても利用可能。
- **Agent Skills**: `.agent/skills/` ディレクトリに `SKILL.md` を配置することで、AI に特定の役割やワークフローを定義できる仕組み。再利用可能な「振る舞いのテンプレート」として機能する。
- **ゼロからのサイト生成**: HTML/CSS/JavaScript の知識がなくても、要件を伝えるだけで完成度の高い静的サイトを生成できた。レスポンシブ対応やアクセシビリティも自動的に考慮される。

## 実践内容
1. Claude Code に「プロフィールサイトを作ってほしい」と依頼
2. プロフィール情報（経歴、スキル、資格など）をコンテキストとして提供
3. AI が HTML/CSS/JavaScript を一括生成し、そのままブラウザで表示可能なファイルを出力
4. Cloudflare にデプロイして公開

## 感想
「コードを書く」のではなく「要件を伝える」だけでプロダクトが出来上がる体験は衝撃的だった。人がコーディングしない時代が現実味を帯びてきたと感じる。一方で、生成されたコードの品質を評価する目は依然として必要であり、「何を作りたいか」を明確に言語化する能力がますます重要になると思う。
